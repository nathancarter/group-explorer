Command-line parameters

So far Group Explorer has only one command-line parameter, documented below

-config <filename>

Examples:

GroupExplorer.EXE -config C:\any\file\on\your\computer.config [Windows]

open GroupExplorer.app -config /any/file/on/your/computer.config [Macintosh]

Instructs Group Explorer to process a different configuration file than the default. This is useful for sharing your configuration file with a friend, for instance. You can email someone your configuration file and that person can use it without erasing his or her own.
